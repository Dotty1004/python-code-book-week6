# 파이썬 코드 도감 (6주차)

학생용 파이썬 코드 도감 자료 저장소입니다.

## 다운로드 방법

### 방법 1: 파일 직접 다운로드
1. 위 파일 목록에서 `파이썬_코드도감_6주차_학생용.html` 클릭
2. 우측 상단의 **Download raw file** 버튼(↓ 아이콘) 클릭
3. 다운로드된 HTML 파일을 더블클릭하여 브라우저에서 열기

### 방법 2: 전체 저장소 ZIP으로 다운로드
1. 저장소 메인 페이지에서 초록색 **Code** 버튼 클릭
2. **Download ZIP** 선택
3. 압축을 풀고 HTML 파일을 더블클릭

## 사용 방법

다운로드한 HTML 파일을 크롬, 엣지, 사파리 등 웹 브라우저에서 열면 바로 학습할 수 있습니다. 별도의 설치가 필요하지 않습니다.
